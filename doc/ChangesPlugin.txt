1. INTRODUCTION                                         *changes-introduction*

The Changes plugin shows changes in your buffer using signs in the sign column.
It can show changes against the last saved version or against version control.

==============================================================================
2. USAGE                                                     *changes-usage*

To enable changes tracking: >
  :EnableChanges
<
To disable: >
  :DisableChanges
<
To toggle: >
  :ToggleChangeView
<

==============================================================================
3. CONFIGURATION                                     *changes-configuration*

You can configure the plugin by calling the setup function: >
  lua require('changes').setup({
    autocmd = true,
    vcs_check = false,
    fast = true,
    -- other options
  })
<

==============================================================================
4. COMMANDS                                           *changes-commands*

:EnableChanges [file]    Enable changes tracking, optionally against [file]
:DisableChanges          Disable changes tracking
:ToggleChangeView        Toggle changes tracking on/off

==============================================================================
5. MAPPINGS                                           *changes-mappings*

]h                      Move to next change
[h                      Move to previous change
ah                      Select a hunk (text object)
